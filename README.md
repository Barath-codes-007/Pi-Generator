# π Digit Generator

A professional π digit generator built with:

- Python
- Flask
- gmpy2
- HTML
- CSS
- JavaScript

## Features

- Generate π to thousands of digits
- Download as TXT
- Download as PDF
- Dark Mode
- Mobile Responsive
- Copy to Clipboard
- Fast Chudnovsky Algorithm
